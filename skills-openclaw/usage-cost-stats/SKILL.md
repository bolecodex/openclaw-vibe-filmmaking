# 用量与费用统计

当用户关心使用本工具的花费、Token 用量或预估费用时，使用本 Skill 进行引导说明。

## 何时使用

- 用户询问「花了多少钱」「费用多少」「用量」「计费」「token 费用」「预估费用」等
- 用户想了解当前项目或会话的消耗

## 回复要点

1. **引导到 OpenClaw Studio**
   - 请用户在 OpenClaw Studio 中打开 **「用量与费用」** 标签页（与「概览」「流水线」等同排）。
   - 该页会展示当前项目的总会话 Token 用量（输入/输出）、预估费用（元），以及按会话的明细。

2. **说明费用性质**
   - 预估费用仅供参考，**实际计费以火山方舟为准**。
   - 官方价格文档：[火山方舟 - 模型价格](https://www.volcengine.com/docs/82379/1544106?lang=zh)

3. **可选补充**
   - 对话侧边栏的 Token 用量条（悬停可看输入/输出/上下文）旁会显示**当前会话**的预估费用。
   - 流水线执行某步骤后，执行日志末尾会显示**本步**的 Token 用量与预估费用。

## 参考链接
