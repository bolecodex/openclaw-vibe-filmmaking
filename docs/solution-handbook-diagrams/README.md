# 对客宣讲 · 矢量架构图（SVG）

本目录为《[解决方案架构师-对客宣讲手册](../解决方案架构师-对客宣讲手册.md)》配套的**矢量示意图**。SVG 可无损缩放，适合 Markdown 预览、导出 PDF、浏览器打开及二次编辑（Figma / Inkscape 等）。

---

## 使用说明

| 场景 | 建议 |
|------|------|
| **转发客户** | 将 `解决方案架构师-对客宣讲手册.md` 与本文件夹**一并打包 ZIP**，避免相对路径断裂。 |
| **投屏讲解** | 正式动画与版式以 **[presentation 门户](../presentation/index.html)** 内 HTML 为准；本目录图为**静态补充**与印刷/邮件附件。 |
| **嵌入文档** | 在 Markdown 中引用：`![说明](./文件名.svg)`；导出 Word / 飞书时确认工具支持内嵌 SVG 或先转 PNG。 |
