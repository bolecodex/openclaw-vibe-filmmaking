本文档基于**火山方舟**相关模型当前公开定价，对「做一部 100 分钟 S 级精品短剧」的 API 费用做测算，供预算与选型参考。实际计费以 [火山方舟模型价格](https://www.volcengine.com/docs/82379/1544106?lang=zh) 为准。

## 二、100 分钟 S 级短剧用量假设

- **成片时长**：100 分钟 = 6000 秒（全部按生成视频计，无库存替换）。
- **剧本与分镜规模**（S 级、精品打磨）：
  - 剧本体量：约 15–20 万字（含对白、旁白、场次说明）。
  - 分镜：按平均约 5 秒/镜，约 **1200 个镜头**；考虑少量合并/拆分，按 **1200 镜头** 计。
  - 角色：主要+配角约 **20 个**，每人至少 1 张立绘，部分 2 张（正/侧等），按 **35 张** 角色图。
  - 场景：独立场景约 **60–80 个**，按 **80 张** 场景参考/氛围图。
  - 分镜配图：**1200 张**（每镜 1 张主图）；重绘、修改、多方案约 **15%** 冗余，即约 **180 张** 额外。
- **文本与 OpenClaw**：
  - 长篇小说→剧本、角色提取、场景切分、分镜生成、对白与剧本多轮编辑、**剪辑相关指令与修改**（分镜剪辑、成片节奏、EDL/切点等）、各类技能调用（含 MCP/工具调用）。
  - 粗估单剧：**输入约 800 万 tokens**，**输出约 400 万 tokens**（含多轮对话与技能调用）；**含剪辑**再预留约 **1.6 倍** 弹性（返工、加戏、审稿、剪辑多轮），即按 **输入 1280 万、输出 640 万** 计。

## 五、与「仅视频」成本的对比

- 仅按「100 分钟 × 1 元/秒」有效成片：**6000 元**。
- 加上抽卡（约 25%）、文本（含剪辑）与生图后，**总 API 费用约 7972–8770 元**（视弹性预留而定），其中：
  - 视频（含抽卡）占比约 **94%**，
  - 生图约 **4%**，
  - 文本 + OpenClaw（含剪辑）约 **2%**。

### 7.2 环境变量配置

在项目根目录的 `.env` 文件中配置（参考 `.env.example`）：

```bash
# 火山方舟 Ark API（图像 Seedream + 视频 Seedance 共用）
ARK_API_KEY=your-ark-api-key
ARK_BASE_URL=https://ark.cn-beijing.volces.com/api/v3

# Seedance 2.0 视频生成
SEEDANCE_MODEL=doubao-seedance-2-0-pro-260215
# 若使用模型单元（EP），优先使用 EP，否则使用 SEEDANCE_MODEL
SEEDANCE_EP=ep-20260316185750-fhsvb
```

**注意**：API Key 为敏感信息，请妥善保管，不要提交到公开版本库。

### 7.3 使用说明

- API Key 用于所有火山方舟 Ark API 调用（包括 Seedream 图像与 Seedance 视频）。
- 若使用模型单元（EP），通常通过 `model` 字段指定 EP ID（如 `ep-20260316185750-fhsvb`），或按具体 API 文档要求配置。
