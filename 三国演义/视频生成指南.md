## 🎬 生成方式说明

### 方式一：使用FFmpeg自动生成（推荐）
#### 依赖：
- FFmpeg 4.0+
- Python 3.7+

#### 生成步骤：
1. 确保所有素材文件都放在正确的目录下：
   ```
   assets/
   ├── characters/  # 角色图像
   ├── scenes/      # 场景图像
   ├── shots/       # 分镜图像（*.jpg，共60个）
   ├── audio/       # 配音文件（dub_001.mp3 ~ dub_060.mp3）
   ├── bgm/         # 背景音乐文件
   └── sfx/         # 音效文件
   ```

2. 使用以下Python脚本读取配置并生成视频：
   ```python
   import yaml
   import subprocess
   import os

   # 读取配置
   with open("视频生成配置_第一回.yaml", "r", encoding="utf-8") as f:
       config = yaml.safe_load(f)

   # 生成FFmpeg命令
   output_path = config["output"]["path"]
   os.makedirs(os.path.dirname(output_path), exist_ok=True)

   # 构建复杂滤镜
   filter_complex = []
   inputs = []
   audio_inputs = []
   current_time = 0

   for scene in config["timeline"]:
       for shot in scene["shots"]:
           # 添加图像输入
           img_path = os.path.join(config["assets"]["shots_dir"], shot["image"])
           inputs.append(f"-loop 1 -t {shot['end_time'] - shot['start_time']} -i {img_path}")
           
           # 添加音频输入
           if "audio" in shot:
               audio_path = os.path.join(config["assets"]["audio_dir"], shot["audio"])
               audio_inputs.append(f"-i {audio_path}")
           
           # 添加BGM输入（如果有变化）
           if "bgm" in shot:
               bgm_path = os.path.join(config["assets"]["bgm_dir"], shot["bgm"])
               audio_inputs.append(f"-i {bgm_path}")

   # 生成命令
   cmd = f"ffmpeg {' '.join(inputs)} {' '.join(audio_inputs)} "
   cmd += f"-filter_complex \"[0:v][1:v]concat=n={len(inputs)}:v=1:a=0[outv]\" "
   cmd += f"-map \"[outv]\" -map 0:a -c:v libx264 -crf 23 -preset medium "
   cmd += f"-c:a aac -b:a {config['output']['audio_bitrate']} "
   cmd += f"-r {config['output']['fps']} -s {config['output']['resolution']} {output_path}"

   print("正在生成视频，请稍候...")
   subprocess.run(cmd, shell=True)
   print(f"视频已生成：{output_path}")
   ```

3. 运行脚本即可自动生成完整视频。

## ⚙️ 自定义调整说明
如果需要调整视频参数，可以直接修改`视频生成配置_第一回.yaml`文件：
1. 修改`output.resolution`可以调整分辨率（如改为"3840x2160"生成4K视频）
2. 修改`output.fps`可以调整帧率（如改为60生成高帧率视频）
3. 修改`transition.duration`可以调整转场时长
4. 修改`text_overlay`相关参数可以调整字幕样式
5. 调整每个shot的`start_time`和`end_time`可以调整镜头时长
